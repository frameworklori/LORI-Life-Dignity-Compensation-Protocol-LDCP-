# LORI-Life-Dignity-Compensation-Protocol-LDCP-
A dignity-first compensation protocol defining a four-stage process—immediate support, moral accountability, safety improvement, and full family-based compensation—to ensure justice beyond cost-benefit calculations.  中 以尊嚴為先的四階段賠償流程：即時援助、道德承擔、安全改善、家庭損失賠償，避免以成本效益衡量生命，確保真正的正義。
